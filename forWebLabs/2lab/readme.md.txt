# Run Node.Js server
**1 step - you need to clone this repo**
```
$ git clone [url]
```
**2 step - install Node**
[Node.js](https://nodejs.org/en/).
**3 step - install node modules**
```
npm install
```
**4 step - run it**
```
npm start
```
